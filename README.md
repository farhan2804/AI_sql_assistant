# AI SQL Assistant

An AI-powered web application that converts natural language into SQL queries using a Large Language Model (LLM). Users can describe what data they want in plain English, and the application generates accurate SQL queries that can be reviewed, copied, explained, and executed against a database.

## Features

* Convert plain English into SQL queries
* AI-powered SQL generation
* Clean and responsive user interface
* Support for multiple SQL dialects (planned)
* Execute generated SQL on a PostgreSQL database (planned)
* Query history (planned)
* AI explanation of generated SQL (planned)
* User authentication (planned)
