# 🤖 AI SQL Assistant

AI SQL Assistant is a Full Stack AI-powered web application that converts natural language into SQL queries using Groq Llama 3.3 and executes them on a PostgreSQL database.

---

## Features

- Convert English to SQL using AI
- Execute SQL on PostgreSQL
- Display results dynamically
- Copy generated SQL
- Loading indicator
- Toast notifications
- Dark / Light mode
- Responsive UI
- Secure execution (SELECT queries only)

---

## Tech Stack

### Frontend
- React
- Tailwind CSS
- Axios
- React Toastify

### Backend
- Spring Boot
- JDBC
- REST APIs

### Database
- PostgreSQL

### AI
- Groq API (Llama 3.3 70B)

---

## Architecture

User → React → Spring Boot → Groq AI → SQL Generation → PostgreSQL → React Table
